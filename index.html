⚙️ Configure o Firebase

Para usar o Nexus em qualquer aparelho com seus dados sincronizados, conecte um projeto Firebase gratuito.

1. Acesse console.firebase.google.com
2. Crie um projeto → adicione um app Web
3. Vá em Firestore Database → criar banco (modo teste)
4. Vá em Authentication → ativar Google
5. Copie as credenciais abaixo e cole nos campos

Precisa de ajuda? Ver documentação Firebase
